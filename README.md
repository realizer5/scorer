# Scorer
*Scorer* is a web application to manage local cricket tournaments.
It includes :
* Tournaments
* Teams
* Players
* Matches

Built with `Next.js`, `Prisma`,`Neon`, `PostgreSQL`, `shadcn/ui`

This project is currently in **MVP Stage**, focused on admin-driven tournament management, with public pages and live scoring

## Features
### Admin Features
* Create Tournaments
* Add Teams to Tournaments
* Add Players to Teams(with roles)
* Create matches (TeamA vs Team B)
* List matches per tournament
* Match management pages
